# A simple Todo app written in FastAPI

**Used Components:**
- FastAPI
- Jinja2
- HTML, CSS, Bootstrap

## Installation and Usage

### 1- Clone the repo:
```sh
git clone https://github.com/Mozart4242/pythonstuff.git
```
### 2- Start the application:
```sh
cd pythonstuff/fastapi/todo_app

uvicorn main:app --reload
```
